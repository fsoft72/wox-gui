Getting Started

1

Install

npm install wox-gui

Or use the CDN — no build step required:

<script type="module" src="https://cdn.jsdelivr.net/npm/wox-gui@0.2.1/dist/wox-gui.cdn.js"></script>
2

Import

// Register all components
import 'wox-gui/register';

// Or import selectively (tree-shakable)
import { WoxButton, WoxInput } from 'wox-gui';
3

Use

<wox-button variant="text" glow>Click me</wox-button>
<wox-input label="Name" placeholder="Enter your name"></wox-input>
